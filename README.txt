Project Overview:
This is a React project using Tailwind CSS for styling and Framer Motion for animations.

Technologies Used:
- React (v19.1.0)
- React DOM (v19.1.0)
- Tailwind CSS (v4.1.8) with Vite plugin (@tailwindcss/vite v4.1.8)
- Framer Motion (v12.16.0)
- React Icons (v5.5.0)

Time Taken:
- Development took approximately 1 day.

Notes:
- The project uses Vite as the build tool.
- Tailwind CSS is configured with the Vite plugin for optimized builds.
- Framer Motion is used to handle smooth animations.
- React Icons provides scalable vector icons for UI.
- Ensure you run `npm install` or `yarn install` to install all dependencies before starting the project.

How to Run:
1. Clone the repository.
2. Run `npm install` or `yarn install`.
3. Run `npm run dev` or `yarn dev` to start the development server.
